# Mental Health Disorder Prediction System

**Mental Health Disorder Prediction System** is a Machine Learning-based application that analyzes symptom patterns to predict potential mental health conditions. The system uses patient-reported behavioral and psychological indicators to classify disorders using trained ML models.

The project demonstrates the **complete Machine Learning workflow**, including data preprocessing, feature engineering, model training, evaluation, and an interactive prediction interface.

---

## Features

* **Machine Learning Diagnosis** : Predicts possible mental health disorders based on symptoms.
* **Multiple ML Models** : Implements Random Forest, Logistic Regression, KNN, and Naive Bayes.
* **Symptom-Based Prediction** : Uses psychological and behavioral indicators as inputs.
* **Interactive GUI Interface** : Allows users to enter symptoms and get predictions instantly.
* **Comparative Model Evaluation** : Identifies the most accurate prediction model.

---

## Disorders the System Can Predict

The system analyzes symptoms across multiple psychological dimensions and predicts conditions such as:

* **Bipolar Disorder (Type 1 & Type 2)**
* **Major Depressive Disorder**
* **Anxiety Disorders (GAD, Panic Disorder, Phobias)**
* **PTSD (Post-Traumatic Stress Disorder)**
* **Schizophrenia Spectrum / Psychosis**
* **Obsessive Compulsive Disorder (OCD)**
* **ADHD / Neurodevelopmental Disorders**
* **Personality Disorders (BPD, NPD, ASPD)**

---

## How It Works

The system evaluates symptoms across different mental health domains:

* **Thought & Perception Disorders** – hallucinations, delusions, disorganized thoughts
* **Mood Disorder Spectrum** – depression, mania, seasonal mood changes
* **Anxiety & Trauma Axis** – panic attacks, PTSD symptoms, excessive worry
* **Neurodevelopmental & Compulsive Patterns** – ADHD, ASD traits, OCD behaviors
* **Personality & Behavioral Patterns** – relational instability, impulsivity, risk-taking

The model processes these features and predicts the most likely mental health condition.

## Installation & Running Locally

1. Clone the repository

```bash
git clone https://github.com/nimrahpunjwani/Mental-Disorder-Prediction-System.git
```

2. Navigate to the project directory

```bash
cd Mental-Disorder-Prediction-System
```

3. Open the notebook

```bash
jupyter notebook
```

4. Run the notebook

Open:

```
mental_health_prediction.ipynb
```

Run the cells to train the model and perform predictions.
